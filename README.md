# PM2.5 Emissions Data Analysis (1999–2008)

This project analyzes fine particulate matter (PM2.5) emissions in the United States using the National Emissions Inventory dataset.

## 📊 Objectives
- Analyze emission trends over time
- Compare emissions across cities
- Study impact of different pollution sources

## 🛠️ Tools Used
- R
- ggplot2

## 📈 Analysis Performed
1. Total PM2.5 emissions in the US
2. Emissions in Baltimore City
3. Emissions by source type (point, nonpoint, on-road, non-road)
4. Coal combustion-related emissions
5. Motor vehicle emissions in Baltimore
6. Comparison of motor emissions: Baltimore vs Los Angeles

## 📌 Key Insights
- Overall emissions decreased from 1999 to 2008
- Different sources contributed differently to pollution trends
- Motor vehicle emissions showed varying trends across cities

## 📁 Files
- `plot1.png` to `plot6.png` → Visualizations
- `plot1.R` to `plot6.R` → Code

## 🚀 Conclusion
The analysis highlights a general decline in PM2.5 emissions over the years, with variations across sources and regions.
